# Site Devenir Distributeur - Duet Studio

Site statique prêt à héberger sur Vercel.

## Déploiement simple

1. Dézipper le dossier.
2. Envoyer le dossier sur GitHub.
3. Importer le repo dans Vercel.
4. Build command : `npm run build`
5. Output directory : `dist`

## En local

```bash
npm install
npm run dev
```

## À modifier rapidement

- `index.html` : textes et structure
- `styles.css` : design
- `script.js` : solutions, tarifs, simulateur
- `assets/` : images et PDF
